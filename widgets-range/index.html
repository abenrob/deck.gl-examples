Range widget

This is an example of a scalable, performant Range Widget using CARTO + deck.gl for a 587k points dataset representing fires worldwide.

This range widget can display and filter the data based on numerical ranges, and it is synchronized with the map using spatialFilter. Learn more about building charts and widgets with CARTO.
