Hello world with Widgets

This is an evolution of our Hello World example where we load a simple dataset of populated places.

We've added two widgets powered by CARTO: a simple scorecard using the getFormula model, and a population histogram using the getHistogram model. Both widgets are filtered by the current map state using spatialFilter, plus a simple country column-based filter.
